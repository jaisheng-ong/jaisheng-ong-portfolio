## Sections

- **About** — Brief introduction with an expandable "Currently" snapshot (what I'm studying, learning, and interested in)
- **Education** — Interactive bubbles for University of Cincinnati and Cincinnati State, each opening a modal with coursework, achievements, and a live degree progress bar
- **Skills** — Three categorized bubbles: Programming Languages, Technologies, and Spoken Languages, each with an icon grid
- **Work Experience** — Detailed cards for each role including Research Intern (MCUT, Taiwan), Software Engineer Intern (Aubot), and earlier experience
- **Projects** — Cards for each project with descriptions, dates, and tech stack icons
- **My Journey** — A dynamic timeline of my full academic and professional path, including co-op availability windows and an expected graduation milestone
- **Contact** — Email, location, and phone displayed in a segmented bar

## Tech Stack

| Layer | Details |
|---|---|
| Structure | HTML5 (single-file, semantic markup) |
| Styling | Embedded CSS with CSS custom properties (light/dark theme variables) |
| Interactivity | Vanilla JavaScript (modals, progress bar animation, smooth scroll, toggle) |
| Icons | Flaticon CDN, devicons (GitHub), git-scm.com |
| Hosting | GitHub Pages |

## Running Locally

No build step required. Clone the repo and open `index.html` directly in any browser:

```bash
git clone https://github.com/jaisheng-ong/jaisheng-ong-portfolio.git
cd jaisheng-ong-portfolio
open index.html
```
